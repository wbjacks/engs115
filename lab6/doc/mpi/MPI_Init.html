MPI_Init

This function initializes, sets up, and generally prepares MPI for your program. You should run MPI_Init in main() before doing anything else -- it should be the first command your program executes.

Prototype:

MPI_Init(int *argc,char ***argv);

You should call MPI_Init as the first command in "main()", and pass it the addresses of argv and argc. MPI_Init will fix argv and argc so that you can check command-line arguments as normal.

MPI_Init should generally be called always and only this way:

main(int argc, char **argv)
{
   int x,y,z;  /* ..Etc.. declare variables here like normal */
   
   MPI_Init(&argc,&argv);

     /* Continue with rest of main() below... */
MPI_Init initializes your program in the same way that MPI_Finalize finalizes it. There's not much else to say.
