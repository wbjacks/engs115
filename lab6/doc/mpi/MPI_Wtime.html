MPI_Wtime

This function returns the time in seconds (and fractions thereof).

Prototype:

double MPI_Wtime();

This routine is useful in timing code. Note that the clocks of different processes are not synchronized and that the usefulness of this routine is in determining elapsed time, not when you should get up in the morning.

This routine is generally used in the following manner:


    double start,
           end,
           time_elapsed;

    ...

    start = MPI_Wtime();

    /* Something you want to time. */

    end = MPI_Wtime();
    time_elapsed = end - start;
